# Session End Skill

## Objective
Properly conclude each Ralph loop iteration.

## End-of-Session Checklist

### 1. Code Quality
- [ ] All new code compiles/runs without errors
- [ ] No obvious bugs or issues introduced
- [ ] Code follows project style guidelines

### 2. Git Status
- [ ] All changes are committed
- [ ] Commit messages are descriptive
- [ ] No untracked files that should be tracked

### 3. Documentation
- [ ] New features/changes are documented
- [ ] Comments added for complex logic
- [ ] README updated if needed

### 4. Task Progress
- [ ] Update plan.md with completed tasks
- [ ] Note any blockers or issues
- [ ] List next priorities

### 5. Session Summary
Write a brief summary including:
- What was accomplished
- Any challenges faced
- Recommended next steps

## Signal Completion
When all tasks for this iteration are done, output:

```
AGENT_DONE

Session Summary:
- [List accomplishments]
- [Note any issues]
- [Suggest next steps]
```

This signals Go Ralph! to end the current iteration.
