# Bookmarks — How to Use

Save and organize reference links, useful URLs, and resources.

## Commands

| Command | What it does |
|---------|--------------|
| `/bookmark-this "title"` | Save a bookmark |

## Workflow

1. Encounter useful reference material
2. Save with `/bookmark-this` for future reference
3. Review bookmarks when tackling related work

## Conventions

- Keep bookmarks organized by topic
- Include context on why the bookmark is useful
- Periodic cleanup of stale bookmarks
