# Lessons — How to Use

Capture and reference lessons learned from development sessions.

## Commands

| Command | What it does |
|---------|--------------|
| `ccs lessons list` | List all lessons |
| `ccs lessons add "Title"` | Add a new lesson |

## Workflow

1. After resolving a complex issue, capture the lesson
2. Include: problem, root cause, solution, prevention
3. Reference lessons when similar issues arise

## Conventions

- Lessons are stored as markdown files
- `LESSONS.md` serves as the main index
- Keep entries concise and actionable
