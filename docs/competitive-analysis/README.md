# Competitive Analysis

This folder contains competitive analysis documents for cosmo-smoke.

## Purpose

- Inform roadmap and feature prioritization
- Understand market positioning
- Document feature inspiration sources
- Track competitor evolution over time

## Contributing

When adding a new analysis:

1. Use naming format: `YYYY-MM-DD-{competitor-name}.md`
2. Include: Overview, Feature Comparison, What We Can Learn, What We Do Better
3. Update this README's table
4. Update comparison-matrix.md if applicable

## Related Commands

```bash
/competitive-analysis https://github.com/user/repo     # Full analysis
/competitive-analysis https://github.com/user/repo --quick  # Summary only
/competitive-analysis --list                            # Show existing analyses
```
