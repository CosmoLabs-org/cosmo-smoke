# Session: Universal Detector + Protocol Assertions

**Date**: 2026-04-21
**Version**: v0.13.0
**Focus**: 22 new project types for universal auto-detection, DNS/SMTP/Docker Compose assertions

## Summary

v0.13.0 expands cosmo-smoke from 9 to 31 detected project types, achieving near-universal language and infrastructure coverage. Alongside the detector expansion, three new wire-protocol assertions (DNS resolution, SMTP ping, Docker Compose health) bring the total assertion count to 32, and the mobile deep link assertion (FEAT-013) was closed out with full documentation.

## Key Changes

- **Universal detector** (22 new project types): Java (Maven/Gradle), .NET/C#, Ruby, PHP, Deno, Scala, Elixir, Swift, Dart, Zig, Haskell, Lua, C/C++ (Make/CMake), Terraform, Helm, Kustomize, Serverless, Hugo, Astro, Jekyll
- **New assertions**: `dns_resolve` (A/AAAA/TXT/MX/CNAME), `smtp_ping` (EHLO handshake), `docker_compose_healthy` (service health status)
- **FEAT-013 closed**: Mobile deep link assertion fully documented and shipped
- **Documentation**: FEATURES.md retroactively updated from v0.6.0 to v0.12.0, CI/CD integration section added to README, universal coverage doc written
