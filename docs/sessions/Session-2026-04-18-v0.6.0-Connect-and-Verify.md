# Session: v0.6.0 — Connect and Verify

**Date**: 2026-04-18
**Version**: v0.6.0
**Focus**: Universal connectivity assertions and pre-commit integration

## Summary

Expanded assertion coverage to answer "does it connect to everything it needs?" Added 4 new assertion types (url_reachable, service_reachable, s3_bucket, version_check), plus WebSocket assertion, monorepo sub-config discovery, gRPC build tag support, and pre-commit hook integration. Used parallel GLM agent execution for independent assertion implementations.

## Key Changes

- Add `url_reachable` and `service_reachable` assertions (shared `httpReachable` helper)
- Add `s3_bucket` assertion (anonymous HEAD, S3-compatible endpoints)
- Add `version_check` assertion (shell command + regex matching)
- Add WebSocket assertion type
- Add monorepo sub-config discovery (`--monorepo` flag)
- Add gRPC health check (build tag: `-tags grpc`)
- Add pre-commit hook integration (`.pre-commit-hooks.yaml`)
- Add `skip_if` conditional execution
- Add env config merge (staging.smoke.yaml)
- Implement Goss-to-cosmo-smoke migration tool (ROAD-024)

## Execution Strategy

Parallel GLM agents for independent assertions (url_reachable, s3_bucket, version_check), sequential for schema structs (dependency) and runner wiring (integration).
