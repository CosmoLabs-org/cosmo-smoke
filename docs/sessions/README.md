# Development Sessions

Session summaries documenting accomplishments, decisions, and next steps.

## File Format

- **Naming**: `Session-NNN-title.md` (sequential)
- Include date and duration
- Document decisions and rationale

## Template

```markdown
# Session NNN: Title

**Date**: YYYY-MM-DD
**Duration**: X hours

## Accomplishments
- [What was completed]

## Decisions Made
- [Important decisions and rationale]

## Problems Encountered
- [Issues faced]

## Next Steps
- [What to do next]
```

## Examples

- `Session-001-initial-setup.md`
- `Session-015-auth-implementation.md`
