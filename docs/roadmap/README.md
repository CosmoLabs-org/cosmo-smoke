# Roadmap

Future plans and feature ideas. Where the project is heading.

## File Format

- **Naming**: `roadmap.md` for main, `quarter-YYYY-QN.md` for quarterly
- Organize by timeframe or priority
- Update regularly as priorities shift

## Contents

- Planned features
- Improvement ideas
- Technical debt items
- Long-term vision

## Examples

- `roadmap.md`
- `2024-Q1-goals.md`
- `technical-debt-priorities.md`
